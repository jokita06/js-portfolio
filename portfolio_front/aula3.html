Arrays em JavaScript

Tipos de Variáveis

let - Variáveis locais, disponíveis apenas no bloco de instrução>

Exemplo:let nome = "João";

const - Valores fixos que não podem ser alterados>

Exemplo:const PI = 3.14;

Arrays

const numeros = [10, 20, 30, 40, 50];
const nomes = ["Ana", "Carlos", "Maria"];

Manipulação de Arrays

push() - Adiciona no final

numeros.push(60); // [10, 20, 30, 40, 50, 60]

pop() - Remove do final

numeros.pop(); // [10, 20, 30, 40]

unshift() - Adiciona no início

numeros.unshift(0); // [0, 10, 20, 30, 40, 50]

shift() - Remove do início

numeros.shift(); // [20, 30, 40, 50]

splice() - Remove/insere em posição específica

numeros.splice(2, 1); // Remove 1 item na posição 2

Métodos Avançados

forEach() - Executa função para cada elemento

numeros.forEach(num => {
                console.log(num);
            });

map() - Transforma array em novo array

const dobros = numeros.map(num => num * 2);

filter() - Filtra elementos

const pares = numeros.filter(num => num % 2 === 0);

reduce() - Reduz a um valor único

const soma = numeros.reduce((total, num) => total + num, 0);
