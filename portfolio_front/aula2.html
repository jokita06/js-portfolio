Funções e DOM em JavaScript

Funções em JavaScript

O que são Funções?

  • Blocos de código reutilizáveis
  • Executam tarefas específicas
  • Tornam o código mais organizado
  • Podem receber parâmetros e retornar valores

Tipos de Funções

1. Função Declarada

function saudacao(nome) {
        return "Olá, " + nome;
    }

2. Função Anônima

const quadrado = function(x) {
        return x * x;
    };

3. Arrow Function

const dobrar = x => x * 2;

DOM (Document Object Model)

O que é o DOM?

  • Representação da estrutura do documento HTML
  • Interface para manipulação da página
  • Permite alterar conteúdo, estrutura e estilo

Métodos para Acessar Elementos

getElementById
getElementsByClassName
getElementsByTagName
querySelector
querySelectorAll
