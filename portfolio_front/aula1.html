Comandos básicos JavaScript

História do JavaScript

  • Páginas web eram estáticas no início (HTML puro)
  • Netscape criou o Livescript (renomeado para JavaScript)
  • Tornou-se a linguagem mais popular para web
  • Suportada por todos os navegadores modernos

Componentes do Front-End

HTML

Estrutura e conteúdo das páginas

CSS

Estilização e aparência visual

JavaScript

Interatividade e dinamismo

Interação com Usuário

Funções Básicas

  • alert() - Exibe mensagens
  • prompt() - Solicita entrada
  • confirm() - Confirmação Sim/Não

Exemplo: let nome = prompt("Qual seu nome?");

Operadores

Aritméticos

+ (adição)
- (subtração)
* (multiplicação)
/ (divisão)
% (módulo)
** (exponenciação)

Relacionais

> (maior)
< (menor)
>= (maior ou igual)
<= (menor ou igual)
== (igual)
!= (diferente)

Lógicos

&& (E)
|| (OU)
! (NÃO)

Estruturas de Controle

Condicionais

  • if/else - Execução condicional
  • switch - Múltiplas condições

Laços de Repetição

  • for - Número conhecido de repetições
  • while - Enquanto condição for verdadeira
  • do...while - Executa pelo menos uma vez

Exemplo for:

for(let i = 0; i < 5; i++) { console.log(i); }
